## Code example
This code changes the layout of the site header if the user has a small screen on the device, or if it reduces the browser window.
```
$(function(){
	if($(window).width() < 420){
		$('.logo-line').after( $('.top_search'));
	}else{
		$('.status').after( $('.top_search'));
	}
	$(window).resize(function() {
		if($(window).width() < 420){
			$('.logo-line').after( $('.top_search'));
		}else{
			$('.status').after( $('.top_search'));
		}
	});
});
```
